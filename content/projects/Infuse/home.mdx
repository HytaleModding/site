Infuse is a mana-based automation mod, heavily inspired by the Create mod.

Everything you need to know about this mod is in this documentation, as well as information about tags and functions for custom plugins.
If you have any question about items, functionalities and future plan for the project, join [our Discord](https://discord.gg/bRDYSVtHvn)!
All original files will be available for free on [Infuse's Github]().

(All info you see in this docu will be updated with the addition of new items and/or functionalities)
