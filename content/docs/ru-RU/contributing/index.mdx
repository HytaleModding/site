# Внести свой вклад в HytaleModding

Мы приветствуем вклад сообщества для улучшения и расширения документации для моддинга. Любой ваш вклад значим для нашего сообщества: Исправление опечаток, добавление руководств, примеры кодов

## Как внести свой вклад?

### Сообщить о проблеме

Если вы нашли проблему с документацией, то, пожалуйста, сообщите об этом, открыв новый репорт (issue) в нашем [GitHub репозитории](https://github.com/HytaleModding/site/issues).

Если же у вас есть возможность решить данную проблему самостоятельно создайте PR.

### Отправка Pull Request'ов

1. Форкнуть репозиторий на GitHub.
2. Перенесите форк на вашу локальную машину.
3. Создать новую ветку для ваших изменений

```bash
git checkout -b my-feature-branch
```

4. Внесите свои изменения и оставьте достаточно ясный комментарий по поводу самого  Мы следуем [общепринятым в нашем сообществе commit](https://www.conventionalcommits.org/en/v1.0.0/), ваш PR не будет принят, если ваши сообщения для commit не будут следовать этим условиям.

```bash
git commit -m "добавьте фичу Х в такую-то документацию"
```

5. Внесите ваши изменения в ваш форк

```bash
git push origin my-feature-branch
```

6. Откройте pull запрос на оригинальном репозитории, опишите причину по которой вы решили внести данные изменения.
