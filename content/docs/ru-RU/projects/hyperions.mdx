# Hyperions

<Callout>
  Это проект сообщества, созданный пользователем [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

**Hyperions** это библиотека для Hytale, созданная, для того чтобы предоставить другим модам API для хранения, передачи и получения энергии. Эта библиотека использует Transfer API (от участника сообщества shailist), и основана на [Energy](https://github.com/TechReborn/Energy) API от команды TechReborn.

Стандартная единица энергии, используемая Hyperions, называется `Гиперион` (Hyperion) (сокращ. `Hy`), а один `Гиперион` примерно равен {/* TODO */}. The standard unit of transfer rate is the Hyperion per tick (`Hy/t`), or alternatively divide by 30 to get Hyperions per second (`Hy/s`).
