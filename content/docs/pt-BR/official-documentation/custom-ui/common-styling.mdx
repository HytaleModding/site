# Introdução

## Biblioteca comum de estilização de interface

Esse documento descreve os componentes de interface e estilos compartilhados definidos em `Common.ui`.

---

### Estilos

Estamos fornecendo alguns estilos que podem ser reutilizados em qualquer interface personalizada, proporcionando uma experiência de interface coesa com a do jogo. Esses estilos estão disponíveis no arquivo `Common.ui`.

Você pode encontrar uma lista com todos os estilos definidos com exemplos em tempo real executando o comando `/ui-gallery` dentro do jogo.

![Gallery](/assets/official-documentation/ui/gallery.png)

### Importando

Para usar esses estilos no seu arquivo de interface:

```
$Common = "Common.ui";

// Em seguida, referencie os estilos e os componentes:
$Common.@TextButton { @Text = "Meu botão"; }
$Common.@Container { ... }
```

<Callout type="info">
O arquivo `Common.ui` está em `Common/UI/Custom/Common.ui` dentro do pacote de Hytale. Se o seu documento de interface personalizado estiver em uma subpasta em `Common/UI/Custom/`, referencie usando a navegação relativa de caminho:

```
$Common = "../Common.ui";
```

Você pode encontrar mais informações sobre caminhos [aqui](markup#path).

</Callout>
