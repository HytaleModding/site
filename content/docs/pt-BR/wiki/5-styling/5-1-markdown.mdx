## Formatação de texto

```md
**Texto em negrito**
*Texto em itálico*
~~Texto riscado~~
**_Negrito e itálico_**
```

## Cabeçalhos

```md
# Cabeçalho 1
## Cabeçalho 2
### Cabeçalho 3
#### Cabeçalho 4
```

## Listas

### Desordenada

```md
- Item
- Item
  - Item aninhado
  - Item aninhado
- Item
```

### Ordenada

```md
1. Primeiro
2. Segundo
   1. Aninhado
   2. Aninhado
3. Terceiro
```

### Lista de tarefas

```md
- [x] Concluído
- [ ] Não concluído
```

## Links

```md
[Texto do link](https://example.com)
[Link com o título](https://example.com "Título de passagem")
```

## Imagens

O site pode conter imagens de recursos externos.

```md
![Texto alternativo](https://example.com/image.png)
![Texto alternativo](https://example.com/image.png "Título da imagem")
```

## Código

### Em linha

```md
Use o `código` dentro do texto.
```

### Bloco

````md
```js
const hello = "mundo";
console.log(hello);
\```
````

Dicas de linguagem suportadas: `js`, `ts`, `css`, `html`, `json`, `md`, `bash` e outras.

## Bloco de citaões

```md
> Isto é uma citação longa.
> Ela pode conter várias linhas.
>
> E vários parágrafos.
```

## Alertas

Os alertas são citações longas estilizadas para destacar informações importantes.

```md
> [!NOTE]
> Informações úteis que o leitor deve saber.

> [!TIP]
> Dicas úteis para fazer as coisas de maneira melhor.

> [!IMPORTANT]
> Informações importantes que o leitor precisa estar ciente.

> [!WARNING]
> Possíveis problemas que o leitor deve ficar atento.

> [!CAUTION]
> Ações que podem causar problemas ou perda de dados.
```

## Tabelas

```md
| Coluna A  | Coluna B  | Coluna C  |
| --------- | --------- | --------- |
| Célula    | Célula    | Célula    |
| Célula    | Célula    | Célula    |
```

Alinhamento:

```md
| Esquerda |  Centro  | Direita  |
| :------- | :------: | -------: |
|  Célula  |  Célula  |  Célula  |
```

## Regra horizontal

```md
---
```

## Notas de rodapé

```md
Um texto com uma nota de rodapé.[^1]

[^1]: Este é o conteúdo da nota de rodapé.
```

## Escapar caracteres

```md
\*Não é itálico\*
\# Não é um cabeçalho
```
