# Visão geral

Este guia explica como adicionar um ícone ao seu mod dentro da aba de configurações de mod do mundo, como mostrado na imagem abaixo.
![mod icon in mod settings tab png](/assets/guides/mod-icon-show.png)

## Passos

1. Prepare o ícone do seu mod. Se você ainda não tem um, você precisará criá-lo.
2. Redimensione o ícone do seu mod para 256x256 e salve o novo arquivo PNG como `icon-256.png`.
3. Mova o arquivo `icon-256.png` para a pasta de recursos do seu mod.

<Files>
    <Folder name="resources" defaultOpen>
        <File name="icon-256.png" />
    </Folder>
</Files>

| Arquivo      | Diretório  | Resolução do Arquivo |
| ------------ | ---------- | -------------------- |
| icon-256.png | resources/ | 256x256              |
