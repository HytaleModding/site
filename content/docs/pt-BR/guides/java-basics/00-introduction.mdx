Bem-vindo(a) à sua jornada em programação Java! Este guia foi projetado para apresentar a você os fundamentos do java.

## O que é Java?

O Java é uma linguagem de programação poderosa e orientada a objetos, criada em 1995. É uma das linguagens de programação mais populares do mundo, usada para tudo, desde aplicativos móveis até software empresarial. E o mais importante para nós, é a linguagem que Hytale usa para seus servidores e API de modificação.

## O que você aprenderá

Este guia foi estruturado para ajudá-lo passo a passo:

1. Sintaxe e fundamentos do Java;
2. Controle de fluxo;
3. Programação orientada a objetos.

Focaremos apenas nos conceitos específicos do Hytale. (A fazer)

## Configurando seu ambiente

Antes de mergulharmos na programação, você vai precisar de algumas ferramentas excrescenciais, você precisa instalar:

### Kit de desenvolvimento Java (JDK)

Baixe e instale o JDK 25 ou superior no [site oficial da Oracle](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html). Se preferir, use uma alternativa de código aberto, como a [AdoptOpenJDK](https://adoptopenjdk.net/).

### Ambiente de desenvolvimento integrado (IDE)

Uma IDE é um aplicativo de software que oferece recursos facilitadores para programadores durante o desenvolvimento de software. Escolhas populares para Java incluem:

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## A mentalidade de criação de mods

A criação de mods é uma resolução criativa de problemas. Você irá:

* Adicionar novos blocos, itens e criaturas para o Hytale;
* Mudar as mecânicas de jogo para criar experiências únicas;
* Construir sistemas que interagem com o mundo do jogo;
* Compartilhar suas criações com a comunidade.
