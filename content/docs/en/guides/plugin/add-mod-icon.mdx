# Overview

In this guide, you'll learn how to add an icon for your mod in the world mod settings tab.
![mod icon show](/assets/guides/mod-icon-show.png)

## Steps

1. Resize your mod icon to 256x256 and save the new png file as `icon-256.png`.
2. Move the icon-256.png file to your resources folder.
