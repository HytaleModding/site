### FieldFunction

Allows you to select which Props to assign based on a Density field and your configured delimiters. The Density’s value at each position determines which delimiter’s Prop to assign.

Parameters:

<table><thead><tr><th width="218.5841064453125">Name</th><th width="205.6929931640625">Type</th><th>Description</th></tr></thead><tbody><tr><td>FieldFunction</td><td>Density slot</td><td></td></tr><tr><td>Delimiters</td><td>list of delimiters</td><td>Attaches Props to value ranges of the Density field.</td></tr><tr><td>Delimiters[].Min</td><td>decimal number</td><td>Lower inclusive bound of the delimiter.</td></tr><tr><td>Delimiters[].Max</td><td>decimal number</td><td>Higher exclusive bound of the delimiter.</td></tr><tr><td>Delimiters[].Assignments</td><td>Assignments asset slot</td><td>The Assignments providing the Props to use.</td></tr></tbody></table>

### Sandwich

Allows you to select which Props to assign based on their vertical (world Y) position and your configured delimiters. Depending on each position’s world height, the matching delimiter’s Prop is assigned.

![](/assets/official-documentation/worldgen/technical-hytale-generator/assignments/sandwich.png)

### Weighted

Picks which Prop to assign randomly based on weights and a seed.

Parameters:

| Name                | Type                               | Description                                         |
| ------------------- | ---------------------------------- | --------------------------------------------------- |
| Seed                | string                             |                                                     |
| SkipChance          | decimal number (0.0–1.0)           | Chance to skip a position without assigning a Prop. |
| WeightedAssignments | list of weighted Assignments slots |                                                     |
| Assignments         | Assignments asset slot             |                                                     |
