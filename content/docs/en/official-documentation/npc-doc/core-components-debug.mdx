# List of Core Components (debug)

## Log: Action (Stable) [#action_log]

Log a message to console.

### Attributes

#### - Once (Stable)

- Execute only once
- **Type**: `Boolean`, **Optional** (Default: false)

#### - Message (Stable)

- Text to print to console.
- **Type**: `String`, **Required**, **Computable**
