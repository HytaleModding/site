## Aperçu

Ceci est une liste complète de **chaque paquet qu'un client peut envoyer au serveur.** Le client envoie des actions abstraites, pas des entrées clavier brutes. Le serveur ne voit jamais "W pressé" - seulement les états de mouvement résultants ou les demandes d'action.
Cette liste n'est pas exclusive aux entrées de touches, mais plutôt à tous les paquets.

Le formatage est paquet.drapeau, c'est-à-dire `MovementStates.idle` ou `ClientPlaceBlock.position`

## Structures de Données

### Enums vs Structs

Les paquets utilisent deux types de structures de drapeaux/valeurs :

| Type       | Structure                                | Taille  | Sérialisation                                    |
| ---------- | ---------------------------------------- | ------- | ------------------------------------------------ |
| **Enum**   | Valeur unique d'un ensemble fixe        | 1 octet | Valeur entière de la constante enum sélectionnée |
| **Struct** | Plusieurs champs envoyés ensemble        | Variable| Chaque champ sérialisé séquentiellement          |

| `flying`          | Vol créatif                      |
