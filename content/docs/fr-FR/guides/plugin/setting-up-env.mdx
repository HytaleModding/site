Ce guide vous accompagnera dans la configuration d'un environnement de développement complet pour le modding Hytale, incluant tous les outils et dépendances nécessaires.

## Prérequis

Avant de commencer, assurez-vous d'avoir :

- Un ordinateur fonctionnant sous Windows 10/11, macOS ou Linux
- Au moins 8 Go de RAM
- 10 Go d'espace disque libre
- Des privilèges administrateur sur votre système

## Logiciels Requis

### 1. Java Development Kit (JDK)

Le modding Hytale nécessite Java 25 ou ultérieur. Nous recommandons d'utiliser OpenJDK.

#### Windows

1. Téléchargez OpenJDK 25 depuis [Adoptium](https://adoptium.net/)
2. Exécutez l'installateur avec les paramètres par défaut
3. Vérifiez l'installation en ouvrant l'Invite de commandes et en exécutant :

```bash
java -version
```

#### macOS

```bash
# En utilisant Homebrew
brew install openjdk@25
```

#### Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install openjdk-25-jdk
```

### 2. Environnement de Développement Intégré (IDE)

Nous recommandons IntelliJ IDEA Community Edition pour le modding Hytale.

1. Téléchargez depuis le [site web de JetBrains](https://www.jetbrains.com/idea/download/)
2. Installez avec les paramètres par défaut
3. Lancez et complétez l'assistant de configuration initiale

### 3) Maven

1. Visitez le site officiel : https://maven.apache.org/download.cgi
2. Téléchargez ce fichier : `apache-maven-3.9.12-bin.zip` (Archive zip binaire)
3. Décompressez-le
4. Ajoutez le dossier `apache-maven-3.9.12/bin/` à votre PATH

Vous pouvez maintenant essayer la commande `mvn -version` pour vérifier que Maven est correctement installé. Assurez-vous de le faire dans un nouveau terminal plutôt que dans un que vous aviez ouvert auparavant !

## Configuration de Votre Espace de Travail

### 1. Ajouter la Dépendance HytaleServer.jar

Avant de pouvoir commencer le modding, vous devez ajouter le fichier HytaleServer.jar comme dépendance :

1. **Téléchargez HytaleServer.jar** en utilisant le [Hytale Downloader](https://support.hytale.com/hc/en-us/articles/45326769420827-Hytale-Server-Manual)
2. **Ajoutez à IntelliJ IDEA** :
   - Cliquez sur **"File"** en haut à gauche
   - Allez dans **"Project Structure"**
   - Allez dans **"Libraries"**
   - Cliquez sur l'**icône +**
   - Sélectionnez le **HytaleServer.jar** que vous avez téléchargé

### 2) Cloner le Template de Plugin

Au lieu de créer un répertoire vide, nous utiliserons le template de plugin Hytale officiel :

```bash
# Clone the template repository
git clone https://github.com/HytaleModding/plugin-template.git MyFirstMod
cd MyFirstMod
```

### 3. Ouvrir dans IntelliJ IDEA

1. Ouvrez IntelliJ IDEA
2. Cliquez sur "Open" et naviguez vers votre répertoire `MyFirstMod`
3. IntelliJ le détectera automatiquement comme un projet Maven
4. Attendez que le projet s'indexe et que les dépendances se téléchargent

## Prochaines Étapes

Maintenant que votre environnement de développement est configuré avec le template de plugin :

1. **Personnalisez le template** - Éditez les fichiers `pom.xml` et `manifest.json` pour changer le nom et les détails de votre mod
2. **Explorez la structure du code** - Familiarisez-vous avec l'organisation du template
3. **Commencez le modding** - Commencez à écrire votre premier plugin Hytale !

Le template de plugin inclut :

- Système de build Gradle pré-configuré
- Structure de plugin d'exemple
- Dépendances essentielles
- Configuration des outils de développement
