Dans ce guide, vous apprendrez comment gérer les inventaires des joueurs dans votre mod Hytale.

## Accéder à l'Inventaire du Joueur

Pour accéder à l'Inventory d'un `Player`, vous pouvez utiliser la méthode `getInventory()` qui retourne l'objet `Inventory`.

```java
Inventory inventory = player.getInventory();
```

## Ouvrir des Inventaires

Vous pouvez ouvrir différents types d'inventaires, ces inventaires sont connus sous le nom de "Pages". Vous pouvez également ajouter des pages personnalisées, cela sera couvert dans un guide ultérieur.

Vous pouvez utiliser l'énumération `Page` pour référencer les inventaires existants, actuellement les pages suivantes sont disponibles :

- `Page.None`
- `Page.Bench`
- `Page.Inventory`
- `Page.ToolsSettings`
- `Page.Map`
- `Page.MachinimaEditor`
- `Page.ContentCreation`
- `Page.Custom`

```java
PageManager pageManager = player.getPageManager();
Store<EntityStore> store = player.getWorld().getEntityStore().getStore();
pageManager.setPage(player.getReference(), store, Page.Inventory);
```

## Classe `ItemStack`

Vous pouvez créer et manipuler des objets dans l'inventaire d'un joueur en utilisant la classe `ItemStack`. Cette classe représente une pile d'objets, et fournit des méthodes pour gérer la quantité et le type d'objets dans la pile.

### Créer un ItemStack

Pour créer un `ItemStack`, vous devez spécifier le type de matériau et la quantité d'objets dans la pile.

```java
ItemStack item = new ItemStack("Stone"); 
ItemStack withQuantity = new ItemStack("Stone", 64);
```

#### Ajouter des métadonnées personnalisées

Vous pouvez également ajouter des métadonnées personnalisées à un `ItemStack` en passant un `BsonDocument` lors de sa création.

```java
BsonDocument metadata = new BsonDocument();
metadata.append("customData", new BsonString("value"));
ItemStack item = new ItemStack("Stone", 64, metadata);
```

### Définir une durabilité

```java
ItemStack stackWithDurability = new ItemStack(
    "DiamondSword", // itemId
    1,              // quantity
    100.0,          // durability
    100.0,          // maxDurability
    metadata        // metadata (optional)
);
```

## Obtenir l'`ItemContainer`

La classe `Inventory` fournit des méthodes pour obtenir plusieurs objets `ItemContainer`, comme :

- `.getStorage()`
- `.getArmor()`
- `.getBackpack()`
- `.getHotbar()`
- `.getUtility()`

Il existe également des méthodes combinées comme :

- `.getCombinedEverything()`
- `.getCombinedArmorHotbarStorage()`
- `.getCombinedBackpackStorageHotbar()`
- `.getCombinedHotbarFirst()`
- `.getCombinedStorageFirst()`
- `.getCombinedArmorHotbarUtilityStorage()`
- `.getCombinedHotbarUtilityConsumableStorage()`

## Ajouter des objets `ItemStack` à l'Inventaire

Pour ajouter un `ItemStack` à l'inventaire d'un joueur, vous pouvez utiliser la méthode `addItemStack()` de la classe `Inventory`.

```java
Inventory inventory = player.getInventory();
inventory.addItemStack(item);
```

Ou vous pouvez spécifier un certain emplacement pour l'ajouter :

```java
inventory.addItemStackToSlot((short) 4, stack)
```

## Retirer des objets `ItemStack` de l'Inventaire

Pour retirer un `ItemStack` de l'inventaire d'un joueur, vous pouvez utiliser la méthode `removeItemStack()` de la classe `Inventory`.

```java
Inventory inventory = player.getInventory();
inventory.removeItemStack(item);
```

Ou vous pouvez spécifier un certain emplacement pour le retirer :

```java
inventory.removeItemStackFromSlot((short) 4);
```
