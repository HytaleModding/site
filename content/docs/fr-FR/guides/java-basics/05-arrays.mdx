---
title: 05 - Tableaux
description: Apprenez à stocker et gérer des collections de données en utilisant les tableaux.
---

Les tableaux vous permettent de stocker plusieurs valeurs du même type dans une seule variable. Pensez à un tableau comme une rangée de boîtes, chacune contenant un élément.

## Créer des tableaux

### Méthode 1 : Déclarer et initialiser séparément

```java
int[] numbers;           // Déclarer
numbers = new int[5];    // Créer un tableau avec 5 emplacements
```

### Méthode 2 : Tout d'un coup

```java
int[] numbers = new int[5];
```

### Méthode 3 : Avec des valeurs initiales

```java
int[] numbers = {10, 20, 30, 40, 50};
```

<Callout type="info" title="Taille du tableau">
  Une fois que vous créez un tableau, sa taille est **fixe**. Vous ne pouvez pas ajouter ou supprimer d'éléments plus tard !

  ```java
  int[] inventory = new int[9];  // A toujours exactement 9 emplacements
  ```
</Callout>

## Accéder aux éléments du tableau

Les tableaux utilisent des numéros d'**index** commençant à **0** :

```java
String[] players = {"Alice", "Bob", "Charlie"};

System.out.println(players[0]); // "Alice"
System.out.println(players[1]); // "Bob"
System.out.println(players[2]); // "Charlie"
```

### Représentation visuelle

| Index | 0       | 1     | 2         |
| ----- | ------- | ----- | --------- |
| Valeur | "Alice" | "Bob" | "Charlie" |

## Modifier les valeurs du tableau

```java
int[] health = {100, 80, 90};

health[1] = 60;  // Changer la santé de Bob à 60

System.out.println(health[0]); // 100
System.out.println(health[1]); // 60
System.out.println(health[2]); // 90
```

## Longueur du tableau

Utilisez `.length` pour obtenir la taille d'un tableau :

```java
int[] scores = {45, 67, 89, 92, 55};
System.out.println(scores.length); // 5
```

<Callout type="warning" title="Length n'est pas une méthode">
  Remarquez qu'il n'y a pas de parenthèses ! C'est `array.length`, pas `array.length()`.
</Callout>

## Parcourir les tableaux

### En utilisant une boucle For

```java
String[] items = {"Épée", "Bouclier", "Potion"};

for (int i = 0; i < items.length; i++) {
    System.out.println(items[i]);
}
```

### En utilisant une boucle For améliorée (For-Each)

```java
String[] items = {"Épée", "Bouclier", "Potion"};

for (String item : items) {
    System.out.println(item);
}
```

<Callout type="info" title="Boucle For améliorée">
  La boucle for améliorée est plus simple et plus sûre, mais vous ne pouvez pas modifier le tableau ni connaître l'index actuel.

  ```java
  // Quand vous avez besoin de l'index
  for (int i = 0; i < items.length; i++) {
      System.out.println(i + ": " + items[i]);
  }

  // Quand vous avez juste besoin des valeurs
  for (String item : items) {
      System.out.println(item);
  }
  ```
</Callout>

## Tableaux multidimensionnels

Les tableaux peuvent contenir d'autres tableaux, créant une grille :

```java
int[][] grid = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

System.out.println(grid[0][0]); // 1
System.out.println(grid[1][2]); // 6
System.out.println(grid[2][1]); // 8
```

### Représentation visuelle

|       | 0 | 1 | 2 |
| ----- | - | - | - |
| **0** | 1 | 2 | 3 |
| **1** | 4 | 5 | 6 |
| **2** | 7 | 8 | 9 |

### Parcourir des tableaux 2D

```java tab="Code"
int[][] grid = {
    {1, 2, 3},
    {4, 5, 6}
};

for (int row = 0; row < grid.length; row++) {
    for (int col = 0; col < grid[row].length; col++) {
        System.out.print(grid[row][col] + " ");
    }
    System.out.println();
}
```

```java tab="Output"
1 2 3
4 5 6
```

## Exemples pratiques

### Système d'inventaire de joueur

```java
String[] inventory = new String[9]; // 9 emplacements de barre d'action

// Ajouter des objets
inventory[0] = "Épée en diamant";
inventory[1] = "Bouclier";
inventory[8] = "Nourriture";

// Afficher l'inventaire
for (int i = 0; i < inventory.length; i++) {
    if (inventory[i] != null) {
        System.out.println("Emplacement " + i + ": " + inventory[i]);
    } else {
        System.out.println("Emplacement " + i + ": Vide");
    }
}
```

### Trouver le score le plus élevé

```java
int[] scores = {45, 92, 67, 88, 55, 71};
int highest = scores[0];

for (int i = 1; i < scores.length; i++) {
    if (scores[i] > highest) {
        highest = scores[i];
    }
}

System.out.println("Score le plus élevé : " + highest);
```

### Calculer la moyenne

```java
double[] temperatures = {23.5, 25.0, 22.8, 24.3, 26.1};
double sum = 0;

for (double temp : temperatures) {
    sum += temp;
}

double average = sum / temperatures.length;
System.out.println("Température moyenne : " + average);
```

### Grille de blocs (Tableau 2D)

```java
String[][] terrain = new String[5][5];

// Remplir avec de l'herbe
for (int x = 0; x < 5; x++) {
    for (int z = 0; z < 5; z++) {
        terrain[x][z] = "herbe";
    }
}

// Placer de la pierre
terrain[2][2] = "pierre";
terrain[1][3] = "pierre";

// Afficher la grille
for (int x = 0; x < terrain.length; x++) {
    for (int z = 0; z < terrain[x].length; z++) {
        System.out.print(terrain[x][z] + " ");
    }
    System.out.println();
}
```

## Opérations courantes sur les tableaux

### Copier un tableau

```java
int[] original = {1, 2, 3, 4, 5};
int[] copy = original.clone();

// Ou manuellement
int[] copy2 = new int[original.length];
for (int i = 0; i < original.length; i++) {
    copy2[i] = original[i];
}
```

### Rechercher une valeur

```java
String[] items = {"Épée", "Bouclier", "Potion", "Arc"};
String target = "Potion";
int foundIndex = -1;

for (int i = 0; i < items.length; i++) {
    if (items[i].equals(target)) {
        foundIndex = i;
        break;
    }
}

if (foundIndex != -1) {
    System.out.println("Trouvé à l'index : " + foundIndex);
} else {
    System.out.println("Non trouvé");
}
```

### Compter les occurrences

```java
String[] blocks = {"pierre", "terre", "pierre", "herbe", "pierre"};
String searchFor = "pierre";
int count = 0;

for (String block : blocks) {
    if (block.equals(searchFor)) {
        count++;
    }
}

System.out.println(searchFor + " apparaît " + count + " fois");
```

## Limitations des tableaux

<Callout type="warning" title="Taille fixe">
  Les tableaux ne peuvent pas grandir ni rétrécir. Si vous avez besoin de flexibilité, vous apprendrez les `ArrayList` plus tard !

  ```java
  int[] numbers = new int[5];
  // Impossible d'ajouter un 6ème élément !

  // Si vous avez besoin de plus d'espace, créez un nouveau tableau
  int[] bigger = new int[10];
  for (int i = 0; i < numbers.length; i++) {
      bigger[i] = numbers[i];
  }
  ```
</Callout>
