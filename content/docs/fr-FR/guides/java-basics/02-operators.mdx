Les opérateurs sont des symboles qui effectuent des opérations sur les variables et les valeurs. Ce sont les verbes de la programmation - ils font bouger les choses !

## Opérateurs arithmétiques

Utilisés pour les calculs mathématiques :

```java
int a = 10;
int b = 5;

int sum = a + b;        // Addition: 15
int difference = a - b; // Soustraction: 5
int product = a * b;    // Multiplication: 50
int quotient = a / b;   // Division: 2
int remainder = a % b;  // Modulo (reste): 0
```

<Callout type="info" title="Note sur la division">
  Lors de la division d'entiers, Java effectue une division entière, ce qui signifie qu'elle supprime toute partie décimale. Par exemple, `7 / 2` donnerait `3`, pas `3.5`.

  ```java
  int x = 7 / 2;       // 3 (pas 3.5!) - Division entière
  double y = 7.0 / 2;  // 3.5 - Au moins un double donne un résultat double
  double z = (double) 7 / 2;  // 3.5 - Le cast fonctionne aussi
  ```
</Callout>

### Modulo (%) - L'opérateur de reste

L'opérateur modulo retourne le reste d'une opération de division :

```java
int remainder = 10 % 3; // 1 (parce que 10 divisé par 3 est 3 avec un reste de 1)
```

## Opérateurs d'assignation composés

Ces opérateurs combinent une opération arithmétique avec une assignation :

```java
int a = 10;

a = a + 5;  // L'ancienne façon

a += 5;     // a = a + 5
a -= 3;     // a = a - 3
a *= 2;     // a = a * 2
a /= 4;     // a = a / 4
a %= 3;     // a = a % 3
```

## Opérateurs d'incrémentation et de décrémentation

Utilisés pour augmenter ou diminuer la valeur d'une variable de 1 :

```java
int a = 5;
a++; // Incrémentation: a devient 6
a--; // Décrémentation: a redevient 5
```

<Callout type="info" title="Préfixe vs Suffixe">
  ```java
  int a = 5;
  int b = a++; // b est 5, a est maintenant 6
  int c = ++a; // a est 7, c est maintenant 7
  ```
</Callout>

## Opérateurs de comparaison

Utilisés pour comparer deux valeurs, retournant un résultat booléen (`true` ou `false`) :

```java
int a = 10;
int b = 5;

a == b // false - Égal à
a != b // true  - Différent de
a > b  // true  - Supérieur à
a < b  // false - Inférieur à
a >= b // true  - Supérieur ou égal à
a <= b // false - Inférieur ou égal à
```

## Opérateurs logiques

Utilisés pour combiner plusieurs expressions booléennes :

```java
boolean x = true;
boolean y = false;

x && y // false - ET logique
x || y // true  - OU logique
!x     // false - NON logique
```

### Combiner les opérateurs logiques

```java
boolean result = (a > b) && (b < 10); // true si les deux conditions sont vraies
```

## Concaténation de chaînes

L'opérateur `+` peut aussi être utilisé pour concaténer (joindre) des chaînes :

```java
String greeting = "Hello, " + "world!"; // "Hello, world!"
String name = "Alice";
String greetingWithName = "Hello, " + name + "!"; // "Hello, Alice!"
```

## Priorité des opérateurs

La priorité des opérateurs signifie quel opérateur est évalué en premier dans une ligne de code. Pensez-y comme BODMAS/PEMDAS en mathématiques !

```java
int result = 10 + 5 * 2; // le résultat est 20, pas 30
```
