Les variables sont des conteneurs qui stockent des données dans votre programme. Pensez à elles comme des boîtes étiquetées où vous pouvez mettre différents types d'informations.

## Qu'est-ce qu'une variable ?

Une variable a trois composants clés :

1. **Type** : Quel type de données elle contient (par ex. entier, chaîne, booléen)
2. **Nom** : L'étiquette que vous utilisez pour vous référer à la variable
3. **Valeur** : Les données qu'elle contient

## Déclarer/Créer des variables

En Java, vous devez déclarer une variable avant de l'utiliser, comme vous auriez besoin d'une boîte avant d'y mettre quelque chose. La syntaxe est :

```java
int age; // Une déclaration
age = 18; // Une assignation

int age = 18; // Déclaration et assignation en une seule ligne, aussi considéré comme une initialisation
```

## Types de données primitifs

Vous vous demandez peut-être ce que signifie « int ». C'est un type de données primitif en Java. Java a 8 types de données primitifs. Voici les plus importants pour le modding :

| Type de données | Description                         | Exemple                    |
| --------------- | ----------------------------------- | -------------------------- |
| int             | Entier (nombres entiers)            | `int score = 100;`         |
| double          | Nombres décimaux                    | `double pi = 3.14;`        |
| boolean         | Valeurs vrai ou faux                | `boolean isActive = true;` |
| char            | Caractère unique                    | `char grade = 'A';`        |
| String          | Séquence de caractères (texte)      | `String name = "Hytale";`  |

## Conversion de type

Parfois vous pourriez ne pas vouloir qu'une variable reste du même type. Vous pouvez convertir entre les types :

### Automatique

```java
int num = 10;
double decimalNum = num; // 10.0 - fonctionne automatiquement
```

### Manuelle (Cast)

```java
double decimalNum = 9.78;
int num = (int) decimalNum; // 9 - les décimales sont supprimées
```

## Constantes

Si vous voulez une variable qui ne devrait pas changer, utilisez le mot-clé `final` :

```java
final double PI = 3.14159;
```

**Convention** : Les constantes utilisent MAJUSCULES_AVEC_UNDERSCORES.

## Pratique

Il est préférable de pratiquer ce que vous venez d'apprendre - ne regardez pas ci-dessus maintenant et essayez de faire ceci vous-même. Si vous êtes bloqué, regardez de nouveau !

1. Déclarez une variable entière nommée `lives` et définissez-la à 3.
2. Déclarez une variable double nommée `gravity` et définissez-la à 9.81.
3. Déclarez une variable booléenne nommée `isGameOver` et définissez-la à false.
4. Convertissez la variable `lives` en double et stockez-la dans une nouvelle variable nommée `livesAsDouble`.
5. Déclarez une constante nommée `MAX_LEVEL` et définissez-la à 100.
