Bienvenue dans votre voyage au cœur de la programmation Java ! Ce guide est conçu pour vous introduire aux bases de Java.

## Qu'est-ce que Java ?

Java est un langage de programmation puissant et orienté objet créé en 1995. C'est l'un des langages de programmation les plus populaires au monde et il est utilisé pour tout, des applications mobiles aux logiciels d'entreprise. Plus important encore pour nous, c'est le langage que Hytale utilise pour ses serveurs et son API de modding.

## Ce que vous allez apprendre

Ce guide est structuré pour vous guider étape par étape à travers :

1. La syntaxe et les bases de Java
2. Le flux de contrôle
3. La programmation orientée objet

Nous nous concentrerons ensuite uniquement sur les concepts spécifiques à Hytale. (à définir)

## Configurer votre environnement

Avant de plonger dans la programmation, vous aurez besoin de quelques essentiels. Vous devez installer :

### Kit de développement Java (JDK)

Téléchargez et installez JDK 25 ou supérieur depuis le [site officiel d'Oracle](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) ou utilisez une alternative open-source comme [AdoptOpenJDK](https://adoptopenjdk.net/).

### Environnement de développement intégré (IDE)

Un IDE est une application logicielle qui fournit des facilités complètes aux programmeurs pour le développement logiciel. Les choix populaires pour Java incluent :

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## Comment utiliser ce guide

1. Lisez chaque article dans l'ordre - Les concepts s'appuient les uns sur les autres
2. Tapez les exemples - Ne copiez-collez pas ; taper vous aide à apprendre
3. Expérimentez - Modifiez les exemples pour voir ce qui se passe
4. Pratiquez - Chaque article inclut des exercices
5. Soyez patient - La programmation est une compétence qui prend du temps à développer

## L'état d'esprit du modding

Le modding est de la résolution créative de problèmes. Vous allez :

* Ajouter de nouveaux blocs, objets et créatures à Hytale
* Changer les mécaniques de jeu pour créer des expériences uniques
* Construire des systèmes qui interagissent avec le monde du jeu
* Partager vos créations avec la communauté
