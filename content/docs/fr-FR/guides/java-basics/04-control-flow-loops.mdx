---
title: 04 - Flux de contrôle - Boucles
description: Apprenez à répéter du code efficacement en utilisant les boucles en Java
---

Les boucles vous permettent de répéter du code sans l'écrire encore et encore. Elles sont essentielles pour traiter des collections, générer des mondes et mettre à jour des objets de jeu.

## Boucle While

Répète du code tant qu'une condition est vraie :

```java tab="Code"

int countdown = 5;

while (countdown > 0) {
    System.out.println(countdown);
    countdown--;
}
System.out.println("Décollage !");
```

```java tab="Output"
5
4
3
2
1
Décollage !
```

### Structure

```java
while (condition) {
    // code à répéter
}
```

<Callout type="warning" title="Boucles infinies">
  Assurez-vous toujours que votre condition de boucle deviendra éventuellement fausse, sinon votre programme tournera indéfiniment !

  ```java tab="Bad Example"
  int x = 0;
  while (x < 10) {
      System.out.println("Bloqué !");
      // On a oublié d'augmenter x !
  }
  ```

  ```java tab="Good Example"
  int x = 0;
  while (x < 10) {
      System.out.println("Comptage : " + x);
      x++; // Ceci augmente x ! // [!code ++]
  }
  ```
</Callout>

## Boucle Do-While

Comme une boucle while, mais s'exécute toujours au moins une fois :

```java
int health = 0;

do {
    System.out.println("Tentative de réapparition...");
    health = 100;
} while (health <= 0);
```

## Boucle For

Idéale quand vous savez combien de fois répéter :

```java tab="Code"
for (int i = 1; i <= 5; i++) {
    System.out.println("Niveau " + i);
}
```

```java tab="Output"
Niveau 1
Niveau 2
Niveau 3
Niveau 4
Niveau 5
```

### Structure

```java
for (initialisation; condition; mise_à_jour) {
    // code à répéter
}
```

<Callout type="info" title="Parties de la boucle For">
  ```java
  for (int i = 0; i < 10; i++) {
  //   ^         ^        ^
  //   Début     Arrêt    Pas
  }
  ```

  * **Initialisation** : int i = 0 - S'exécute une fois au début
  * **Condition** : i < 10 - Vérifiée avant chaque boucle
  * **Mise à jour** : i++ - S'exécute après chaque boucle
</Callout>

## Instruction Break

Vous pouvez sortir d'une boucle prématurément avec `break` :

```java

for (int i = 1; i <= 10; i++) {
    if (i == 6) {
        break; // Sortir de la boucle quand i est 6
    }
    System.out.println(i);
}

```

## Instruction Continue

Passer à l'itération suivante avec `continue` :

```java
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue; // Sauter les nombres pairs
    }
    System.out.println(i); // Affiche uniquement les nombres impairs
}
```

## Boucles imbriquées

Vous pouvez mettre des boucles à l'intérieur d'autres boucles :

```java
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 2; j++) {
        System.out.println("i: " + i + ", j: " + j);
    }
}
```

<Callout type="warning" title="Performance des boucles imbriquées">
  Faites attention avec les boucles imbriquées ! Elles peuvent ralentir significativement votre mod.

  ```java
  // Ceci s'exécute 1 000 000 de fois !
  for (int i = 0; i < 1000; i++) {
      for (int j = 0; j < 1000; j++) {
          // Code ici
      }
  }
  ```

  Essayez d'éviter les boucles profondément imbriquées lors du traitement de grandes quantités de données.
</Callout>
