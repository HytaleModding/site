---
title: 03 - Flux de contrôle - Instructions If
description: Apprenez à contrôler le flux de vos programmes Java en utilisant les instructions if.
---

Les programmes doivent prendre des décisions. Les instructions if permettent à votre code de choisir différents chemins selon des conditions.

## Instruction If de base

La forme la plus simple vérifie une condition :

```java
int health = 20;

if (health > 10) {
    System.out.println("Vous êtes en bonne santé !");
}
```

### Structure

```java
if (condition) {
    // code à exécuter si la condition est vraie
}
```

## Instructions If-Else

Choisir entre deux chemins :

```java
int health = 5;

if (health > 10) {
    System.out.println("Vous êtes en bonne santé !");
} else {
    System.out.println("Vous avez besoin de soins !");
}
```

## Chaîne If-Else Else-If

Vérifier plusieurs conditions en séquence :

```java

int health = 7;

if (health > 15) {
    System.out.println("Vous êtes en pleine forme !");
} else if (health > 5) {
    System.out.println("Vous allez bien.");
} else {
    System.out.println("Vous avez besoin de soins !");
}
```

## Instructions If imbriquées

Vous pouvez placer des instructions if à l'intérieur d'autres instructions if :

```java
int health = 12;

if (health > 10) {
    if (health > 15) {
        System.out.println("Vous êtes en pleine forme !");
    } else {
        System.out.println("Vous êtes en bonne santé !");
    }
} else {
    System.out.println("Vous avez besoin de soins !");
}
```

## Conditions composées

Vous pouvez combiner plusieurs conditions en utilisant des opérateurs logiques :

```java

// ET - les deux doivent être vraies
int health = 12;
boolean hasPotion = true;

if (health > 10 && hasPotion) {
    System.out.println("Vous êtes en bonne santé et avez une potion !");
} else if (health > 10) {
    System.out.println("Vous êtes en bonne santé mais avez besoin d'une potion !");
} else {
    System.out.println("Vous avez besoin de soins !");
}

// OU - au moins une doit être vraie

int health = 8;
boolean hasPotion = false;
if (health > 10 || hasPotion) {
    System.out.println("Vous êtes soit en bonne santé soit avez une potion !");
} else {
    System.out.println("Vous avez besoin de soins et d'une potion !");
}

// NON - inverse la condition

int health = 8;
if (!(health > 10)) {
    System.out.println("Vous avez besoin de soins !");
} else {
    System.out.println("Vous êtes en bonne santé !");
}
```

## Opérateur ternaire

Une façon compacte d'assigner des valeurs :

```java
int health = 12;
String status = (health > 10) ? "En bonne santé" : "A besoin de soins";
```

### Syntaxe

```java
variable = (condition) ? valeurSiVrai : valeurSiFaux;
```
