CurseForge est une plateforme d'hébergement de mods, vous permettant de partager vos mods et créations Hytale avec les joueurs via notre site web et notre application.

## 1. Créer un Compte CurseForge

Tout d'abord, [visitez notre page de connexion](https://curseforge.com/api/v1/auth/login?returnUrl=https://www.curseforge.com/) et connectez-vous à un compte existant ou cliquez sur "Create an account".

Cela peut être fait en utilisant n'importe quel compte Google, Discord, GitHub ou Twitch.

## 2. Créer un Projet

Maintenant que vous êtes connecté, vous pouvez visiter notre Console Auteur et naviguer vers l'onglet Projects, puis cliquer sur "Create a Project", soit dans la barre latérale soit en haut de la page.

![CurseForge Author Page](/assets/guides/curseforge_author_page.png)

Cela vous amènera ensuite au flux de [création de projet](https://authors.curseforge.com/#/projects/create/choose-game). Tout d'abord, sélectionnez votre jeu (dans ce cas, Hytale) et la page sera mise à jour avec de nouveaux champs en attente de vos entrées. Veuillez consulter nos

[Politiques de Modération](https://support.curseforge.com/en/support/solutions/articles/9000197279-moderation-policies#:~:text=General%20CurseForge%20Moderation%20Policies,-Policy\&text=The%20description%20can%20include%20descriptive,the%20game%22%20is%20not%20enough) pour plus d'informations sur ce qui est autorisé ou non lors de la création d'un nouveau projet. Ci-dessous se trouvent les champs avec leurs descriptions :

- **Name:** C'est le titre de votre projet. Il doit être unique à votre projet. Si le nom est déjà pris, il sera rejeté.
- **Summary:** C'est un court résumé du mod qui apparaîtra dans les listes de projets
- **Description:** C'est l'explication plus longue de ce que fait/ajoute votre mod. Assurez-vous d'élaborer clairement sur votre projet. Nous rejetterons votre projet si la description n'est pas grammaticalement correcte ou si elle ne décrit pas suffisamment le projet que vous créez. Cela peut être modifié plus tard. Note : Tous les titres et descriptions doivent être en anglais.
- **Project License:** C'est un champ déroulant qui contient les licences les plus courantes utilisées dans les jeux de modding. Recherchez le type de licence que vous souhaitez utiliser, y compris ses avantages et inconvénients. Vous pouvez également choisir d'utiliser une Custom License en bas de la liste déroulante et entrer le texte que vous voulez.
- **Class:** C'est la catégorie racine de votre projet. Assurez-vous de sélectionner la bonne, car elle peut impacter les types de fichiers supportés et le support de l'application dans CurseForge.
  - Pour Hytale, il existe quatre types de Class :
    - Mods
      - C'est la classe qui contient à la fois les Asset Packs et les Plugins, et vous devriez télécharger l'un ou l'autre ici.
    - Worlds
    - Prefabs
    - Bootstrap
      - C'est la classe qui contient tous les Early Plugins.
- **Main Category:** C'est la catégorie principale dans laquelle votre projet sera trié.
- **Additional categories:** Vous pouvez en ajouter jusqu'à 4. Votre projet apparaîtra également dans les listes où les gens recherchent ces types de mods. Attention cependant, si vous ajoutez simplement des catégories qui n'ont rien à voir avec votre mod, nous pourrions vous le renvoyer pour modifications.
- **Logo Image:** Une icône unique pour identifier votre projet.

## 3. Télécharger un Fichier

Une fois que vous avez configuré votre projet et rempli les informations ci-dessus, vous devez maintenant télécharger un fichier vers votre projet pour qu'il entre dans la file de modération. Pour ce faire, visitez l'onglet "Files" dans votre page Console Auteur.

![CurseForge Add Project](/assets/guides/curseforge_add_project.png)

Ensuite, cliquez sur "Add File" et téléchargez le fichier de votre mod, en entrant toute information supplémentaire ou métadonnées que vous souhaitez inclure concernant ce fichier.

Le format de fichier dépendra de la classe que vous sélectionnez.

### Pour les Plugins

Si vous téléchargez un plugin pour Hytale, veuillez vous assurer de sélectionner la classe "Mods" et de télécharger votre fichier en tant que fichier jar.

### Pour les Packs

Si vous téléchargez un Pack pour Hytale, veuillez vous assurer de sélectionner la classe "Mods" et de télécharger votre fichier en tant que fichier ZIP

### Pour les Worlds

Si vous téléchargez un World pour Hytale, assurez-vous de le télécharger dans la classe "Worlds" et de télécharger votre fichier en tant que fichier ZIP.

**Une fois que vous avez téléchargé votre fichier, il sera envoyé à notre équipe de modération pour examen, et une fois approuvé, il sera en ligne pour tous les joueurs sur CurseForge !**
