# Mode Créatif

Le mode créatif dans Hytale est un mode bac à sable qui permet aux joueurs d'éditer le monde librement.

## Outils d'édition du monde

Le mode créatif inclut une variété d'outils pour aider les joueurs à éditer le monde. Il inclut une fonctionnalité d'annulation pour les modifications.

Notez que les sections ci-dessous n'étaient pas nommées dans la démo vidéo des outils.

---

### Pointeur laser

Affiche un rayon depuis la caméra du joueur dans la direction où le joueur regarde.

#### Paramètres

- Durée (Curseur entier) : Description inconnue en décembre 2025.
- Couleur du laser (Code couleur hexadécimal) : Description inconnue en décembre 2025.

### Outil pinceau de sculpture

Permet à l'utilisateur de relever et abaisser le terrain pour sculpter et fusionner les formes plus précisément. Supporte la plupart des fonctionnalités du pinceau de peinture.

### Outil pinceau de ligne

Inconnu en décembre 2025.

### Outil de sélection

Permet à l'utilisateur de sélectionner et éditer des régions du monde en transformant des sélections de blocs ou un prefab via la position, la rotation et l'extrusion.

#### Paramètres

- Mode (Menu déroulant : Set, Wall, Fill, Replace) : Description inconnue en décembre 2025.
- Matériau (Curseur entier) : Description inconnue en décembre 2025.
- Poids (Curseur entier) : Description inconnue en décembre 2025.
- Pitch (Entier) : Description inconnue en décembre 2025.

### Règle

Permet à l'utilisateur de mesurer des distances entre plusieurs points dans l'espace 3D, avec support pour plusieurs lignes de mesure et édition des points mesurés.

### Outil pinceau de couche

Inconnu en décembre 2025.

### Outil de collage

Permet à l'utilisateur de coller une sélection copiée de blocs ou un prefab au point où le joueur regarde. Le collage est relatif au point d'ancrage de la sélection. Si la sélection est un prefab, le point d'ancrage peut être défini dans l'éditeur de prefabs. On ne sait pas si le point d'ancrage de la sélection peut être défini autrement.

### Outil pinceau de remplissage

Inconnu en décembre 2025.

### Outil pinceau de peinture

Les pinceaux sont utilisés pour faire de grands changements au monde depuis loin.

#### Capacités générales

- Placer des formes et des prefabs
- Remplacer des blocs
- Créer des verrous d'axe et des masques

#### Paramètres

- Matériau
- Largeur (Curseur entier) : Description inconnue en décembre 2025.
- Hauteur (Curseur entier) : Description inconnue en décembre 2025.
- Forme (Menu déroulant : Cube, Sphère, Cylindre, Cône, Cône inversé, Pyramide, Pyramide inversée, Grille de cuboïdes) : Description inconnue en décembre 2025.
- Origine (Menu déroulant : Centre, Bas, Haut) : Description inconnue en décembre 2025.
- Axe miroir (Menu déroulant : Aucun, autres options inconnues) : Description inconnue en décembre 2025.
- Épaisseur du mur (Curseur entier) : Description inconnue en décembre 2025.
- Densité (Curseur entier) : Description inconnue en décembre 2025.

#### Préréglages :

- Rocher : Crée un rocher au point où le joueur regarde.
- Chemin herbeux : Transforme le terrain en chemin herbeux au point où le joueur regarde.
- Gravats : Génère des gravats au point où le joueur regarde.
- Tentacule : À déterminer
- Fissures de lave : Transforme le terrain en fissure de lave au point où le joueur regarde.
- Sources chaudes : À déterminer
- Ruines : Génère une ruine au point où le joueur regarde.
- Toits de ruines : Génère une ruine avec un toit au point où le joueur regarde.
- Herbe : À déterminer
- Montagne : Place des morceaux consécutifs de terrain pour former une montagne au point où le joueur regarde.
- Ruines : À déterminer

### Machinima

Machinima est un éditeur de séquences intégré, permettant à l'utilisateur de créer et éditer des séquences d'actions à rejouer en jeu. Il inclut un éditeur de keyframes interactif intégré.

La vitesse de lecture peut être ajustée via des commandes.

L'utilisateur peut ajouter des acteurs que la séquence contrôlera.

- Référence
- Caméra
- Entité
- Objet
- Joueur

## Commandes

Les commandes sont entrées dans la console de chat du monde et commencent par une barre oblique `/`.

- `set <bloc>` : permet à l'utilisateur de définir les blocs dans une sélection avec le bloc spécifié.
- `inventory clear` : permet à l'utilisateur de vider son inventaire.
