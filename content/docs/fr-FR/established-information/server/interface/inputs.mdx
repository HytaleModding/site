Hytale est conçu pour supporter une variété de méthodes d'entrée afin d'accommoder différentes préférences de joueurs et plateformes.

## Système d'interaction

Hytale dispose d'un système d'interaction robuste qui permet aux plugins de personnaliser comment les raccourcis clavier sont interprétés et quelles actions ils déclenchent. Ce système est piloté par les données, permettant aux moddeurs de créer des interactions complexes via des assets.

Les moddeurs peuvent définir des raccourcis clavier personnalisés pour diverses actions, comme utiliser des capacités ou interagir avec des objets dans le monde de jeu. Le système d'interaction est conçu pour être puissant, bien qu'il puisse nécessiter un certain effort pour gérer la complexité des assets impliqués.

> Vous pouvez utiliser vos propres interactions pour utiliser des raccourcis clavier pour tout ce que vous voulez. Le système d'interaction est très puissant mais nous devons encore travailler sur son utilisabilité car les assets peuvent devenir assez complexes et difficiles à gérer.
