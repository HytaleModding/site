# Scripting visuel

Hytale prévoit de supporter le scripting visuel comme méthode pour rendre la logique avancée et le flux de contrôle accessibles à un public plus large. Ce système ne sera pas disponible au lancement, mais devrait être disponible peu après.

## Aperçu

Le scripting visuel dans Hytale est conçu pour fonctionner de manière similaire aux Blueprints dans Unreal Engine. Il permet aux concepteurs de créer des comportements complexes via une interface visuelle qui correspond à leur façon de penser la logique du jeu, sans avoir besoin d'écrire du code directement.

### Fonctionnalités clés

* **Sous-ensemble des fonctionnalités Java** : Le scripting visuel peut faire un sous-ensemble de ce qui est possible avec les plugins Java
* **Adapté aux concepteurs** : « Les concepteurs sont responsabilisés grâce à un langage visuel qui correspond à leur façon de penser les comportements »
* **Extensible par les programmeurs** : Les programmeurs peuvent étendre et optimiser le système en ajoutant de nouveaux nœuds visuels et des implémentations sensibles aux performances
* **Environnement de partage sûr** : Fournit un environnement sandboxé et sûr pour partager la logique entre moddeurs

## Avantages par rapport au codage traditionnel

Le scripting visuel rend l'écriture de logique personnalisée plus accessible en supprimant les obstacles courants rencontrés lors du développement en Java :

* **Pas de configuration d'environnement complexe** : Pas besoin de s'inquiéter de l'installation des bons outils ou de la configuration des environnements de développement
* **Développement sans syntaxe** : Supprime le besoin d'apprendre et de retenir la syntaxe de programmation
* **Partage plus sûr** : Les malwares dans le scripting visuel devraient être pratiquement impossibles
* **Plus inclusif** : Abaisse la barrière à l'entrée pour les non-programmeurs

## Relation avec Java

Le scripting visuel et les plugins Java sont conçus pour fonctionner en tandem :

* **Java étend le scripting visuel** : Les programmeurs peuvent créer de nouveaux nœuds visuels en utilisant Java
* **Optimisation des performances** : Bien que le scripting visuel ne soit pas aussi optimisé que Java directement, les programmeurs peuvent implémenter la logique critique pour les performances en Java et l'exposer comme nœuds visuels
* **Le meilleur des deux mondes** : Permet à la majorité de la logique de rester dans l'environnement accessible du scripting visuel tout en gardant les opérations coûteuses optimisées

Hytale recommande fortement le scripting visuel car il est à la fois plus accessible et plus sûr à partager et télécharger. Cependant, rester purement en Java fonctionnera toujours pour ceux qui le préfèrent.

## Scripting visuel 3D

L'une des fonctionnalités uniques de Hytale est d'amener le scripting visuel directement dans le monde 3D :

* **Édition dans le monde** : Liez des leviers, des portes, des générateurs, des déclencheurs et d'autres éléments directement dans le jeu
* **Création de cartes d'aventure** : Excellent pour construire des cartes d'aventure et des rencontres sans quitter Hytale
* **Similaire au SnapMap de DOOM** : Fournit un workflow familier pour ceux qui ont utilisé des systèmes similaires
* **Entièrement intégré** : Fonctionne parfaitement avec la pile d'assets et de scripting

## Pas de scripting textuel

Hytale a pris la décision délibérée de ne pas inclure de langages de scripting textuels. Leur raisonnement :

* **Faux compromis** : Le scripting textuel est souvent introduit pour « aider les non-programmeurs » mais nécessite toujours l'apprentissage de concepts de programmation
* **Augmentation de la complexité** : Les programmeurs finissent par jongler avec deux langages dont l'un est moins puissant
* **Nécessite toujours de la syntaxe** : Les concepteurs doivent toujours apprendre la syntaxe et les concepts de programmation
* **Moins inclusif** : Ne résout pas vraiment le problème d'accessibilité
* **Fragmentation** : Évite de « fragmenter la logique à travers plusieurs "demi-langages" »

Au lieu de cela, Hytale croit que le scripting visuel est une meilleure approche qui rend vraiment la logique du jeu accessible à un public plus large tout en maintenant la puissance et la flexibilité nécessaires.
